POINTERS FOR BLOCK ELEMENTS-DIV

~Width of a block level element is 100% of the viewport width
~Height of a block isn't set by default
~Height and width can be explicitly set using CSS
This is span

POINTERS FOR INLINE ELEMENTS-SPAN

~Width of an inline element is dependent on the content
~Height of an inline isn't set by default
~Height and width cannot be explicitly set using CSS because it depends on the content.
